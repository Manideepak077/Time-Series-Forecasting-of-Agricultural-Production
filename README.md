# *Time Series Forecasting of Agricultural Production*

## Project Overview
This project focuses on forecasting agricultural production using time-series analysis techniques. It compares statistical and deep learning models to identify the most accurate method.

## Objectives
- Analyze historical agricultural data  
- Build forecasting models  
- Compare model performance  
- Identify best model  

## Models Used
- Naïve Model/ Baseline model  
- ARIMA Model  
- LSTM Model  

## Evaluation Metrics
- MAE- Mean Absolute Error
- RMSE- Root Mean Square Error
- MAPE- Mean Absolute Percentage Error 

## Limitations
- Limited dataset  
- No external variables  
- Annual data only  

## Future Scope
- Add weather data  
- Use monthly data  
- Try hybrid models  

## Technologies
Python, Pandas, NumPy, TensorFlow, Keras, Statsmodels  

## How to Run
```bash
pip install -r requirements.txt  
python main.py
git clone https://github.com/Manideepak077/Time-Series-Forecasting-of-Agricultural-Production
```
